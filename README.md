# API de Lista de Itens

API REST simples desenvolvida em Node.js com Express.

## Funcionalidades
- Listar itens
- Criar itens
- Remover itens

## Tecnologias
- Node.js
- Express
- CORS

## Deploy
- Render (Web Service)

## Endpoint base
https://meu-projeto-backend-28d4.onrender.com

## Exemplo de uso
GET /items  
POST /items  
DELETE /items/:id
